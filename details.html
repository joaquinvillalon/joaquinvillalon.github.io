Detalles de App Fotos

Funciones principales:

  • Crear cuenta
  • Subir foto de perfil
  • Publicar fotos desde la cámara o galería
  • Editar fotos
  • Borrar fotos

Secciones disponibles:

  • Sección global: Visualiza las fotos de todos los usuarios.
  • Sección privada: Ve las fotos que has publicado.

Tecnologías utilizadas:

  • Firebase: Base de datos en tiempo real
  • EmailJS: Envío de correos electrónicos
  • Ionic: Desarrollo de la aplicación
  • Android Studio: Generación del APK
  • HTML, CSS y JavaScript: Desarrollo del sitio web
  • GitHub: Repositorio del proyecto
